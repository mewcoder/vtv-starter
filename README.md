## 特性

- ⚡️ Vue3.2, Vite3, TypeScript

- ✅ Vitest
- 🗂 Vue-Router4

## 使用

### GitHub 模板

[使用这个模板创建仓库](https://github.com/mewcoder/vtv-starter/generate).

### 克隆到本地

```bash
npx degit mewcoder/vtv-starter my-vtv-app
cd my-vtv-app
pnpm i
```
